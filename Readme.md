# Spring Boot E-Commerce Store

A full-featured backend for an e-commerce application built with **Spring Boot**, **Spring Data JPA**, and **MySQL**.  
It provides secure user authentication, product management, cart functionality, and order handling.

---

##  Features

- User registration and authentication (JWT-based)
- Role-based access control (Admin, User)
- Product management (CRUD)
- Shopping cart management
- Order creation and tracking
- Address management
- Exception handling and validation
- ModelMapper integration for DTO mapping

---

##  Tech Stack

- **Java 17**
- **Spring Boot 3**
- **Spring Security**
- **Spring Data JPA**
- **Hibernate**
- **MySQL**
- **Lombok**
- **ModelMapper**
- **Maven**
